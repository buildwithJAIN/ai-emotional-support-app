# AI Emotional Support App 🤖❤️

An AI-powered emotional support mobile application built using React Native and modern AI APIs.

## 🚀 Features
- AI-generated emotionally intelligent responses
- Multi-language support
- Voice & text interaction
- Firebase integration
- Clean and scalable architecture

## 🛠 Tech Stack
- React Native (Expo)
- Firebase (Firestore, Auth)
- OpenAI / Gemini API / ElevenLabs API
- JavaScript

## 📌 Purpose
This application aims to provide respectful, supportive, and emotionally aware AI assistance to users in real time.
